# 在选项卡中显示内容

## 具有非共享选择状态的选项卡

### 代码

```markdown
:::tabs
== tab a
a content
== tab b
b content
:::
```

### 示例

:::tabs
== tab a
a content
== tab b
b content
:::

## 具有共享选择状态的选项卡

### 代码

```markdown
:::tabs key:ab
== tab a
a content
== tab b
b content
:::

### 示例

:::tabs key:ab
== tab a
a content 2
== tab b
b content 2
:::
```

:::tabs key:ab
== tab a
a content
== tab b
b content
:::
